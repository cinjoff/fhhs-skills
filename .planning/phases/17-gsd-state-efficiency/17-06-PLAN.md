<objective>
Convert gsd-tools.cjs from eager require() of all 12 lib modules to lazy require() per command.
Currently every invocation loads ~6,600 LOC regardless of which command runs.
Lazy loading reduces per-invocation overhead by only parsing the module(s) needed.
No behavioral changes — purely a startup performance optimization.
</objective>

<context>
@bin/gsd-tools.cjs — the CLI router (lines 139-152 are the eager requires, lines 189-638 are the switch)
@bin/lib/core.cjs — always needed (provides `error()` used before command dispatch)
</context>
