# Broken Project

A TypeScript API project that contains intentional code issues for testing review and analysis skills.

## Vision

Demonstrate a realistic codebase with common mistakes: type errors, unhandled async operations, SQL injection vulnerabilities, and missing error handling.

## Scope

- REST API with Express
- Database query layer (intentionally vulnerable)
- External API calls (intentionally unsafe)

## Goals

- REQ-1: User authentication endpoint
- REQ-2: Database-backed item storage
- REQ-3: Third-party data fetch integration
