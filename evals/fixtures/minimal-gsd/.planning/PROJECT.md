# Minimal GSD Test Project

A simple TypeScript Express API for testing GSD skills.

## Vision

Provide a minimal but realistic project structure that GSD skills can analyze, plan against, and build upon without requiring complex setup.

## Scope

- REST API with Express and TypeScript
- Basic CRUD endpoints for a resource
- JSON responses, no database layer needed

## Out of Scope

- Authentication
- Database integration
- Frontend UI
