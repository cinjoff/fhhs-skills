# Roadmap

## Phase 1: Foundation

**Goal:** Set up project scaffolding and basic Express server.

### Requirements

- REQ-1.1: Initialize TypeScript project with strict mode
- REQ-1.2: Create Express server with health check endpoint
- REQ-1.3: Add basic request logging middleware

## Phase 2: API Endpoints

**Goal:** Implement core CRUD endpoints.

### Requirements

- REQ-2.1: GET /items — list all items
- REQ-2.2: POST /items — create a new item
- REQ-2.3: DELETE /items/:id — delete an item by ID
