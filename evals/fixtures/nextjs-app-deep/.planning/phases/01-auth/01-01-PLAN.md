# Phase 01 Plan: Authentication System

## Objective
Implement JWT-based authentication with login form, token management, and session persistence.

## Tasks

### Task 1: JWT Token Utilities
- **Files:** `src/lib/auth.ts`
- **Action:** Implement `signToken()`, `verifyToken()`, `refreshAccessToken()`, `saveSession()`, `getSession()`, `clearSession()`
- **Done:** Token utilities exist with sign, verify, refresh, and session management

### Task 2: Auth API Routes
- **Files:** `src/app/api/auth/login/route.ts`, `src/app/api/auth/refresh/route.ts`
- **Action:** Login route using Supabase auth, refresh route for token rotation
- **Done:** Both routes handle auth flows and return tokens

### Task 3: Login Page + Auth Hook
- **Files:** `src/app/login/page.tsx`, `src/hooks/use-auth.ts`
- **Action:** Login form with validation, useAuth hook for state management
- **Done:** Login form validates and submits, hook manages auth state
