## Step 1: Welcome

fhhs-skills provides a unified workflow for software development:

- **Composite commands** тАФ `/plan`, `/build`, `/fix`, `/refactor`, `/verify`, `/resume`, `/research`, `/verify-ui`
- **Engineering discipline** тАФ TDD, two-stage code review, evidence-based verification, fresh subagents, YAGNI
- **Design quality** тАФ `/critique`, `/polish`, `/normalize`, `/harden`, `/animate` for frontend work
- **Project tracking** тАФ GSD state management with phases, milestones, roadmaps

Everything is built into this plugin. No external plugins to install.
